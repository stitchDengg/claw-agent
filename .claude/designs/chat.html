React 性能优化讨论
如何优化 React 应用的渲染性能？
CLAW AGENT

优化 React 性能通常需要从渲染机制、资源加载和状态管理三个维度入手。以下是针对性建议：

01

利用 React.memo 避免不必要的重渲染

通过对组件进行记忆化处理，仅在 Props 发生实质性变化时触发更新。这对于大型列表或复杂可视化组件尤为重要。

02

使用 useMemo 和 useCallback

缓存计算量大的结果和回调函数引用，防止在每次渲染时重新创建对象，从而打破子组件的渲染优化。

03

虚拟滚动 (Virtual Windowing)

对于长列表，仅渲染当前视口可见的元素。可以使用 react-window 等库来实现。

component.tsx 
import React, { memo } from 'react';

const HeavyComponent = memo(({ data }) => {
  return (
    <div>
      {/* 复杂渲染逻辑 */}
      {data.map(item => <span key={item.id}>{item.val}</span>)}
    </div>
  );
});

export default HeavyComponent;
